# Architecture

## Основные слои

- `app/` — точки входа приложения.
- `core/` — конфигурация, логирование, общие сервисы.
- `providers/` — внешние ИИ-провайдеры: OpenRouter, OpenAI, Gemini, Anthropic, Ollama.
- `council/` — логика Совета ИИ: оркестрация, голосование, консенсус.
- `crypto/` — будущая интеграция с биржами и техническим анализом.
- `documents/` — будущая обработка PDF, Word, Excel и изображений.
- `plugins/` — расширения.
- `ui/` — будущий графический интерфейс.

## Правило безопасности

Файл `.env` не должен попадать в GitHub. Для примера используется только `.env.example`.
